# VibeCheck
A simple JavaScript-based quiz app that compares answers to see how well two people vibe with each other, based on everyday preferences and choices.
Each matching answer increases the score, giving a fun and simple way to measure compatibility or “vibe match.”
✨ Features

Interactive multiple-choice quiz

Score calculation based on answer matching

Casual, real-life questions (no right or wrong answers)

Event-driven JavaScript logic

Clean and beginner-friendly code structure

Tech Stack

HTML

CSS

Vanilla JavaScript

This project was created to:

Practice JavaScript event-driven programming

Understand state management without frameworks

Build a fun utility to check how well two people vibe with each other
